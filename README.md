# Graw CI

This package contains assets to build a web application that follows the Corporate Identity of Graw Radiosondes.

## Contents

- CSS Styles
    - Colors
        - as root css variables
        - as helper classes with additional shades
            - accent
            - bg
            - border
            - caret
            - decoration
            - outline
            - text
    - Fonts
- Images
    - Icon
    - Logo
    - Favicon
    - PWA Assets

## Installation

```bash
npm install @graw/ci
```

## Development

```bash
npm install
npm run watch
```
